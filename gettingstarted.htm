Getting Started

Welcome to WebAuthor!


This guide will help you get started in using WebAuthor.


The WebAuthor Interface

The WebAuthor interface is comprised of the Markup View, and the various property docks.

The Markup View

This is where you'll create your HTML page structure.

Property Docks

These component allows you to access the style and script properties of your webpage. These also allows you to "Dock" them to the window edge.
Docks in WebAuthor
  • Layout Preview Dock - previews your HTML structure
  • CSS Dock - edit your webpage's stylesheet
  • JS Dock - edit your webpage's Javascript code

Layout Preview

This docked window ( on the right ), is the "Layout Preview" pane, it allows you to see your webpage's HTML structure without having to reload your browser over and over again. However, you can only preview HTML structure ( no styles / scripts ) in the Layout Preview Dock. If you want to see a "Full-featured " preview, you may use the "Preview in Firefox" button under the File menu.

WebAuthor File Options

You can save / open projects in WebAuthor as ".wbproj" files. And once you've finished creating your page, you can also export them in the standard ".htm / .html" format.


Previewing in a Browser

If you want to see what your page would look like on an actual browser, you may use the "Preview in Browser" button to open your page in Mozilla Firefox ( OS X / Linux ) or Internet Explorer on Windows.

Productivity Shortcuts

If you prefer to not let your hands leave the keyboard while working, then you'd be happy to know that WebAuthor allows you to work with a variety of time-saving keyboard shortcuts.

    Interface

  • Toggle Preview Dock ( Ctrl + Shift + 1 )
  • Toggle CSS Dock ( Ctrl + Shift + 2 )
  • Toggle Javascript Dock ( Ctrl + Shift + 3 )

    • Switching

  • Switch to HTML Code ( Ctrl + 1 )
  • Switch to CSS Code ( Ctrl + 2 )
  • Switch to Javascript Code ( Ctrl + 3 )

    • Misc

  • Preview Page ( F5 )
